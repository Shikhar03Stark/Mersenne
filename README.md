# Mersenne
Smart Recommender System for Friends and Group to battle boredom

## Setup Server (dev)

 - Paste the credential file in `/Server/.env`
 - Run `cd Server/`
 - Run `npm install`
 - Run `npm run dev`

## Populate Server/Hotdata

 - Run `cd Server/`
 - Run `NODE_ENV=production node server.js`
